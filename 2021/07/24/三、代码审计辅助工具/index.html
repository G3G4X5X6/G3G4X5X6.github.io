三、代码审计辅助工具

本文最后更新于：2022年2月12日 下午

代码审计过程中或多或少会使用各种辅助工具，选择合适的工具可以起到事半功倍的效果。

0x00 代码编辑器

0x0000 Sublime

官网：https://www.sublimetext.com/

0x0001 IDEA

官网：https://www.jetbrains.com/idea/

1. 快捷键

  • 全局搜索：Shift + Shift
  • Run Anything: Ctrl + Ctrl
  • 格式化代码：Ctrl + Alt + L
  • 快速生成代码：Alt + Insert
  • 重写/实现方法：Ctrl + O
  • 类层次结构： Ctrl + H
  • 方法层次结构：Ctrl + Shift + H
  • 跳转去声明：Ctrl + B / Ctrl + 单击
  • 跳转去实现：Ctrl + Alt + B

0x01 测试工具

0x0100 Burpsuite

官网：https://portswigger.net/burp

0x0101 SwitchyOmega

SwitchyOmega是一款代理管理插件，支持Firefox和Chrome浏览器，并支持HTTP、HTTPS、socket4和socket5协议。

0x0102 Max HackerBar

HackerBar是Firefox的一个插件，也是信息安全从业者常用的经典工具。

0x0103 Postman

官网：https://www.postman.com/downloads/

0x0104 Postwomen

官网：https://hoppscotch.io/cn

0x0105 Tamper Data

Tamper Data是Firefox浏览器的一款Web安全测试插件，它的主要功能包括以下几种：

  • 查看、修改 HTTP/HTTPS 的请求头和请求参数
  • 跟踪 HTTP 请求/响应并记时
  • 对 Web 站点进行安全测试

0x0106 Ysoserial

官网：https://github.com/frohoff/ysoserial

Ysoserial是一款开源的Java反序列化测试工具，内部集成有多种利用链，可以快速生成用于攻击的代码。

0x0107 Marshalsec

官网：https://github.com/mbechler/marshalsec

Marshalsec是一款开源的Java反序列化测试工具，不仅可以生成各类反序列化利用链，还可以快速启动恶意RMI服务等。

0x0108 MySQL监视工具

  • MySQL日志查询工具
  • MySQL Monitor
    • MySQL Monitor是Web版本的SQL记录实时监控工具

0x0109 Beyond Compare

官网：https://www.scootersoftware.com/

0x02 反编译工具

0x0200 JD-GUI

官网：https://github.com/java-decompiler/jd-gui

0x0201 FernFlower

官网：https://github.com/Konloch/bytecode-viewer

0x0202 CFR

官网：https://github.com/leibnitz27/cfr

0x0203 IntelliJ IDEA

官网：https://www.jetbrains.com/idea/

IntelliJ IDEA反编译工具能够自动解包已添加依赖的 Jar 包，并对其内容进行反编译。

0x03 Java 代码静态扫描工具

0x0300 Fortify SCA

官网：https://www.microfocus.com/zh-cn/products

0x0301 VCG

VisualCodeGrpper简称VCG，它是基于VB开发的一款Windows下的白盒审计工具。

0x0302 FindBugs与FindSecBugs

FindBugs是一款Bug扫描插件，在IDEA和Eclipse中都可以进行安装。

0x0303 SpotBugs

SpotBugs是FindBugs的继任者。

0x04 公开漏洞查找平台

0x0400 CVE

官网：http://cve.mitre.org/

Tips: CVE Search 网站

0x0401 NVD

官网：https://nvd.nist.gov/

NVD为美国国家通用漏洞数据库。

0x0402 CNVD

官网：https://www.cnvd.org.cn/

CNVD是中国国家信息安全漏洞平台。

0x0403 CNNVD

官网：http://www.cnnvd.org.cn/

CNNVD是中国国家信息安全漏洞库。
